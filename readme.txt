PrismPay for WooCommerce extends the functionality of WooCommerce to accept payments from credit/debit cards on your checkout page using PrismPay Integration.

== Description ==

<h3>PrismPay for WooCommerce</h3> makes your website ready to use PrismPay Payment Gateway, to accept credit/debit cards on your e-commerce store checkout page.

<h3>WooCommerce 3.x Compatible</h3>

== Features ==

1. Accept Credit/Debit Cards right on your website.
2. No redirecting on other url.
3. Easy to install and configure.
4. Option to configure success & failure message.
